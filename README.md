## Projects Included

### Caesar Cipher

A command-line encryption and decryption application that implements the classical Caesar Cipher algorithm. Users can encrypt or decrypt text messages by specifying a shift value while preserving spaces, numbers, and special characters.

**Key Concepts:**

- String Manipulation
- Modular Arithmetic
- Cryptography Fundamentals
- User Input Validation
- Functions and Control Flow

---

### Password Generator

A command-line application that generates strong and randomized passwords based on user-defined numbers of letters, digits, and special characters. The program uses Python's built-in randomization techniques to create secure passwords.

**Key Concepts:**

- Randomization
- String Manipulation
- Lists
- User Input Handling
- Algorithm Design
